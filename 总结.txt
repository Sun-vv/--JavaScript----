创建一个电子邮件掩码器
1.获取字符串中某个字母的索引号位置 email.indexOf('@')  如果值为-1，则没有找到
2.获取字符串中某个片段 email.slice(start,last)
3.从某个位置开始获取字符串 email.slice(start)
4.获取字符串第一个字符 email[0]
5.重复某个字符 "*".repeat(数字)
function maskEmail(email){
  //1.
  const atIndex = email.indexOf('@');
  if(atIndex === -1){
    return email;
  }

  const userId = email.slice(0,atIndex);  //2.
  const usermain = email.slice(atIndex);  //3.

  if(userId <= 2){
    return email;
  }


  const first = userId[0];  //4.
  const last = userId[userId.length - 1]
  const stars = "*".repeat(userId.length - 2)  //5.

  return first +  stars + last + usermain;

}

原语数据类型和非原语数据类型

原语数据类型是 JavaScript 中最简单的数据形式。它们包括 number、bigint、string、boolean、null、undefined 和 symbol。
这些类型被称为“原语”，因为它们表现单一值且不是对象。
let num1 = 5;
let num2 = num1;
num1 = 10;

console.log(num2); // 5
//原语值（5）从 num1 赋值到 num2


原生类型的数据类型则更复杂。这些是对象，包括普通对象、数组和函数。与原语不同，
非原生类型可以作为属性或元素包含多值。
当你创建一个非原生类型值的变量时，存储在变量中的实际上是对象存储的内存位置的引用，而不是对象本身。
const originalPerson = { name: "John", age: 30 };
const copiedPerson = originalPerson;

originalPerson.age = 31;

console.log(copiedPerson.age); // 31
//因为 originalPerson 和 copiedPerson 都在引用内存中的同一个对象。
//两个变量共享相同的内存位置

补充：split()
/\s+/ 是一个正则表达式，其中：
 \s 表示任何空白字符，包括空格、制表符（\t）、换行符（\n）等。
   + 表示一个或多个（前面的 \s 出现至少一次）。
因此 split(/\s+/) 的作用是：按一个或多个空白字符作为分隔符来拆分字符串。


单词计数方法（按空格分割）：
javascript
function getWordCount(sentence) {
    // 去除首尾空格，再按一个或多个空白字符分割
    const words = sentence.trim().split(/\s+/);
    // 如果原句子全是空格，words[0] 会是空字符串，此时应返回 0
    return words[0] === '' ? 0 : words.length;
}

练习    交换数组的两个元素
splice() 方法可以通过在任意位置（包括中间）添加或删除元素来修改数组。
由于 splice() 返回一个包含被删除元素的数组，你可以使用它在不改变原数组的情况下交换数组中的两个元素。

const originalArray = [12, 97, 68, 55];
const copyArray = originalArray.slice();
copyArray[1] = copyArray.splice(3, 1, copyArray[1])[0];
console.log(copyArray); // [12, 55, 68, 97]
//splice(3, 1, copyArray[1]) 删除索引为 3 的元素（55）

它将来自 copyArray[1]（97）的元素插入到索引 3。

splice() 返回 [55]，一个包含被移除元素的数组

[0] 从该数组中提取 55

该值（55）被赋值回 copyArray[1]，完成交换
